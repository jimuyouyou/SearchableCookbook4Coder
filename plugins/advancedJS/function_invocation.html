There are actually four different ways to invoke a function, each with their own nuances. They are:
  • 1. As a function, in which the function is invoked in a straightforward manner. function ninja(){}; ninja();
  • 2. As a method, which ties the invocation to an object, enabling object-oriented programming. var o = {}; o.whatever = function(){}; o.whatever();
  • 3. As a constructor, in which a new object is brought into being. function do(){} var d = new do();
  • 4. Via their apply() or call() methods
