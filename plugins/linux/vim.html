Common command for vim/vi

url: http://yannesposito.com/Scratch/en/blog/Learn-Vim-Progressively/ http://blog.csdn.net/arganzheng/article/details/6260690
move hjkl h-left, j-down, k-up, l-right
move to document, line, word positions 
ctrl + f: page down
ctrl + b: page up

gg: move to home of file
G: move to end of file
19G: move to the 19th lines of file

0: move to home of line
$: move to end of line

^: move to first character of line
g_: move to last character of line

t.: move to .
copy, cut and paste 
y: copy
d: cut or delete
p: paste
P: paste before this line

yy: copy current line
dd: cut or delete current line
3yy: copy 3 lines
7dd: cut or delete 7 lines

x: delete current character
dw: delete current word
cw: delete current word and go to insert mode

d0: delete to start of line
d$: delete to end of line
y0: copy to start of line
y$: copy to end of line
visual commands 
>: tab to left
<: tab to right
y: copy marked text
d: delete marked text
~: switch case
visual block mode 
0 + (ctrl+v) + (ctrl+d) + I//[ESC]: insert // for selected lines
editing insert 
u: undo
.: repeat last command

i: insert before
I: insert before home of current line
a: insert after
A: insert after end of current line

100ihelloworld: insert 100 helloworld
search replace 
/: search - n for next, N for previous
s/old/new/g: replace all old with new in current line
g/old/new/: replace all old with new in file
others 
ctrl + p: code completion
v: enter visual mode for selection 

ddp: move current line content to next line
ddkP: move current line content to before previous line
