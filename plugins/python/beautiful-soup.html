在windows上安装BeautifulSoup（for Python IDEL）

1、在 http://www.crummy.com/software/BeautifulSoup/ 下载BeautifulSoup，如BeautifulSoup-3.2.1。

2、解压缩到本地硬盘上，如C:\Python25下。

3、运行cmd，执行：

cd C:\Python25\BeautifulSoup-3.2.1

setup.py build

setup.py install

4、在IDEL下import BeautifulSoup 不报错则安装成功。
